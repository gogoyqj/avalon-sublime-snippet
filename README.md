sublime-snippet for avalon

### 安装

```
    sh mac-install.sh 
```

### 使用

```
    # avalon
    vmodel + tab        => 创建一个新版的vmodel，输入变量之后，+ tab进入下一步
    avalon.define + tab => 在变量之后创建一个新版的vmodel，+ tab进入下一步


    # mmRouter相关
    router + tab        => 创建一个router规则
    router.config + tab => 创建错误跳转 + 调用avalon.history

    # mmState相关
    avalon.state + tab  => 创建一个状态 avalon.state ...
    state + tab         => 创建一个状态 .state
    state.config        => 创建全局配置 + 调用avalon.history

    # AMD
    define + tab        => 创建一个AMD模块
```
